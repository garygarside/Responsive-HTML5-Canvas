Responsive HTML5 canvas with draggable object

A little experiment to see how easy it was to make a canvas and a child object relative. Uses a lot of JavaScript to do the math and is currently pretty restricted to some static calculations to rework the position and size of the rectangle. A little extra work would be involved to make multiple dynamic elements respond. Try dragging the rectangle and resizing your browser. Should work in IE9+ and recent builds of Chrome, Firefox, Safari & Opera.
